 # ResQFlow – Real-Time Disaster Response Web Application

ResQFlow is a web application designed to assist disaster response by enabling coordination of relief resources and sharing emergency information in real time.

## Features
- Real-time information sharing
- Coordination of disaster relief resources
- Emergency response guidance

## Technologies Used
- React
- TypeScript
- Tailwind CSS
- Vite

## Purpose
This project explores how modern web technologies can support disaster management systems and improve coordination during emergency situations.
